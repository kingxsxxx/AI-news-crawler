# AI News Terminal - 启动说明

## 快速启动

### 方式一：双击启动（推荐 ⭐）

#### 已构建版本（生产模式）
双击 **`启动应用(已构建).bat`** 文件即可启动应用。

#### 开发版本
双击 **`启动应用.bat`** 文件即可启动。

---

## 手动创建桌面快捷方式

### 方法 A：使用右键菜单
1. 找到可执行文件：
   ```
   C:\Users\浅笑如初\Desktop\Newsagregator\src-tauri\target\release\ai-news-aggregator.exe
   ```
2. 右键点击 `ai-news-aggregator.exe`
3. 选择 **"发送到"** → **"桌面快捷方式"**
4. 重命名快捷方式为 "AI News Terminal"

### 方法 B：在桌面直接创建
1. 在桌面空白处右键点击
2. 选择 **"新建"** → **"快捷方式"**
3. 在"请键入项目的位置"输入框中输入：
   ```
   C:\Users\浅笑如初\Desktop\Newsagregator\src-tauri\target\release\ai-news-aggregator.exe
   ```
4. 点击"下一步" → 输入名称：**AI News Terminal**
5. 点击"完成"

## 开发模式 vs 生产模式

| 模式 | 启动命令 | 优点 |
|------|----------|------|
| 开发模式 | `npm run tauri:dev` | 支持热重载，修改代码后自动刷新 |
| 生产模式 | 直接运行 exe | 启动快，性能更好，无需命令行 |

## 故障排除

### 应用无法启动？
- 检查 Node.js 是否已安装：`node --version`
- 检查 Rust 是否已安装：`cargo --version`
- 运行 `npm install` 确保依赖已安装

### 找不到可执行文件？
- 确认已运行 `npm run tauri:build` 构建
- 检查路径：`C:\Users\浅笑如初\Desktop\Newsagregator\src-tauri\target\release\`
